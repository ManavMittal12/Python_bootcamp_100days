# What are application programming interface (APIs) ?

# there are a lot of data on the websites.
# what if we want to use the data that they have and use it in our own programs ?
# that's where application programming interface(API) comes in

# API is a set of commands, functions, protocols and objects that programmers
# can use to create software or interact with an external system.


# we are going to try and interact with websites and pull data from
# those websites.

# THe API is an interface or rather a sort of barrier between your
# program and an external system. We try to use the rules that the API
# prescribed to make a request to the external system for some peice of
# data. And if you have structured your request as per all the requirements
# the external system set out, then they will respond to you appropriately and give you the data that
# you want. but if you don't follow the rules, the request will be invalid.



# Example
# website are like Restaurant and the data that powers these websites as the
# kitchen behind the scenes.
# in restaurant, we have a menu, that menu is like the interface between
# us and the restaurant, it tells us what we can order and what we can't.

# we can maybe get a cake, sandwich etc. from the restaurant, but we
# can't go into the kitchen and demand whatever we want.

# API is essentially a menu, which tells us what all we can do with an external
# system such as a website that carries data.


# One of the most important aspects of API is the API endpoint, We can
# understand it as a location, for example, if we want to get a peice of data
# from a particular external service, then we need to know where that data is stored.

# it's like wanting to withdraw money from the bank, we need to know where the
#  bank is and what is its address.

# In API, it's called API endpoint
# eg
# api.coinbase.com

# we also have to make a request on the internet, (API Request)
# e.g. taking money out from the bank, there's a process to it too.
# eg, there's a bank-teller that ask you what you want and prevent you from
# going to the vault directly. API does the same thing.
